Dico 2.0 Sur GIT

git init à utiliser au début du projet, initialise.

git commit enregistrer les modifications dans le référentiel

git status affiche l'état du "tree"

git commit enregistrer les modifications dans le référentiel

git commit -m "commentaire" créer les versions du projet

git rm nom_du_fichier supprimer un fichier du répertoire de travail et de l'index

git bisect repérer où se trouve le bug

git blame savoir qui à mit le bug

Branch

git branch permet de lister les branches

git branch "" permet de créer une nouvelle branche
